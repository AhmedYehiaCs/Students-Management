This is my first full project where I combined different pieces of knowledge into a working application. It’s a console-based program that manages student records and stores them in a JSON file for persistence. The app allows you to add new students with details like name, age, hobby, and field, edit existing records, search for a student by name, delete individual students, or clear all records at once. I also practiced handling user input, validating data, and giving clear feedback messages. The project helped me understand how to structure code, work with files, and build a simple but functional system.
